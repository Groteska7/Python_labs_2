# Вариант 8
## класс Sersev взят из ЛР1
## Цель работы
- Научиться работать с коллекциями объектов.
- Понять разницу между моделью сущности и контейнером объектов.
- Реализовать собственный контейнерный класс.
- Освоить итерацию по объектам.
- Реализовать базовые операции управления коллекцией.
### Коллекция позволяет:
- хранить несколько серверов
- проверять работу и задачи конкретных серверов
- сортировать серверы по задачам
- работать с коллекцией серверов как с обычным списком
## Атрибуты класса
- `_services` — список объектов Course, хранящихся в коллекции
### Основные методы
- реализован CRUD
- сортировки: sort_by_task и sort-by_max_task
- поиск: find_by_index и find_by_name
### Получение данных
- `__len__` возврает количество сервисов
- `__getitem__` возвращает все элементы в виде списка если не указан индекс, или элемент по индексу
- `__iter__` делает итерируемым
### логические операции
- `get_by_par(idle|working|mainstance|error)` - возвращает все сервера, состояние которые указано в параметре
### Магические методы
- `__str__` — красивый вывод коллекции
- `__repr__` — техническое представление
